# Black Marker Media LLC

Central hub for research, planning, automation, and growth strategy across all BMB business units.

## Structure

| Folder | Purpose |
|--------|---------|
| `01-research/` | Market research, promotion strategies, scaling insights |
| `02-business-plans/` | Marketing and growth plans ready for implementation |
| `03-tools-and-automation/` | Scripts and workflows to automate business tasks |
| `04-agents/` | AI agents for research, marketing, and operations |
| `05-reporting-and-analytics/` | Performance reports and data analysis |
| `06-financials/` | Revenue forecasts and budget planning |
| `07-opportunities/` | Future opportunities under evaluation and development |
