## TENT & PHYSICAL SETUP

### Canopy/Tent
- [ ] 10x20 pop-up canopy (fire-retardant certified — required by NYC for commercial events)
- [ ] Sidewalls: bring at least 2 sidewalls (back + one side for wind/privacy for tech station)
- [ ] Sandbags: minimum 40–50 lbs per leg × 4 legs (NYC requirement — no staking on asphalt)
- [ ] Tent frame inspected — all connectors secure before day of

### Layout Plan
```
[SIDEWALL - BACK]
|   [TECH STATION - Laptop + Encoder + Audio]   |
|                                                |
|   [COMMENTARY TABLE - 6ft]                    |
|   [Host 1] [Joselito] [Host 2 / empty seat]   |
|                                                |
|   [DJ BOOTH - side or corner]                 |
[OPEN FRONT — CROWD FACING]
```
- Commentary table faces the crowd/street — people walking by see the broadcast in action
- Cameras are positioned so crowd is visible in the background of the stream (street presence on camera)
- DJ booth off to the side — line out to encoder, not mic'd through PA

### Furniture & Branding
- [ ] 6ft folding table (commentary table)
- [ ] Chairs × 3 (Host 1, Joselito, Host 2)
- [ ] Branded tablecloth (BlackMarker.TV)
- [ ] Vertical banners × 2 — one each side of tent opening
- [ ] Overhead banner across front of canopy (event title or BlackMarker.TV)
- [ ] Monitor/display × 1 (optional: show stream playback so crowd outside can see it)

## POWER SETUP

- [ ] Generator: Honda EU3000iS (3,000W inverter) — quiet enough for tent environment
- [ ] Fuel: filled + 5-gallon backup can
- [ ] Generator positioned minimum 10–15 ft from tent rear, vented, away from crowd
- [ ] Heavy-duty outdoor extension cord (30A, 25–50ft) from generator to tent
- [ ] Power strip × 2 with surge protection inside tent
- [ ] Load check: DJ setup (~1,200W) + encoding laptop (~300W) + displays (~400W) + misc (~300W) = ~2,200W — within generator headroom
- [ ] Test full load before event day

## LIVE STREAM TECH SETUP

### Connectivity
- [ ] Cellular bonding device configured (LiveU Solo+ or Teradek Bond)
- [ ] SIM cards active: AT&T + T-Mobile + Verizon
- [ ] Test upload speed at location day before if possible (or test morning of, before crowds)
- [ ] Target: 10+ Mbps stable upload; stream at 6 Mbps for 1080p30
- [ ] Backup plan: iPhone hotspot on secondary carrier if bonding device fails

### Production Software
- [ ] vMix Pro loaded on production laptop
- [ ] Scene layout configured: Commentary cam (main), PIP with field cam, lower thirds, bumper
- [ ] Stream key entered for BlackMarker.TV RTMP
- [ ] Stream key entered for YouTube Live
- [ ] Stream key entered for Facebook Live
- [ ] vMix recording enabled (always record locally — safety net if stream drops)
- [ ] Backup: OBS on backup laptop as failsafe encoder

### Cameras
- [ ] Primary camera on tripod behind table — framing: hosts at table with tent/street behind
- [ ] Camera connected via HDMI to capture card → laptop
- [ ] Field camera charged (ZV-1 or GoPro on gimbal) — Host 2 / field reporter carries this
- [ ] Field cam streaming plan: either RTMP from a phone hotspot to vMix, or Host 2 returns to tent for on-camera segments
- [ ] Camera angles tested, white balance set, exposure locked

### Audio
- [ ] Rode Wireless GO II — transmitter on Joselito, transmitter on Host 1
- [ ] Windscreens on all transmitters
- [ ] Receivers connected to production laptop audio input
- [ ] IEM monitoring: both hosts can hear each other + program audio in the crowd
- [ ] DJ line out: DI box → XLR → audio interface → laptop (confirm DJ has a mixer with line out)
- [ ] Ambient mic set: stereo condenser placed at front of tent
- [ ] Full audio mix tested before go-live: commentary levels, DJ music levels, ambient blend
- [ ] Backup wired mics + XLR cables on standby
