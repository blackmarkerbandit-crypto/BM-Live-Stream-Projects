Paid Ads — Minimal Budget, Gap-Fill Only
Paid ads are the supplement, not the strategy. With 11 days to event and a tight budget, every organic channel should be exhausted first. Use paid spend only to (1) amplify posts that are already performing organically, and (2) retarget people who visited blackmarker.tv/FE3 but didn't buy. Do not build cold-audience campaigns from scratch at this stage.
Recommended Paid Budget Split
Facebook / Instagram Post Boost (amplify best organic posts) ~70%
Facebook / Instagram Retargeting (blackmarker.tv/FE3 visitors who didn't buy) ~30%
Google Ads  — Skip for this event. Insufficient runway to build effective campaigns in 11 days. $0
YouTube Ads  — Skip. No time to build audience pools and test creative. $0
Paramount+ / CTV  — Skip. Minimum $500, too slow for organic events at this timeline. $0
Facebook / Instagram Boost — How to Use It

Only boost posts that are already getting organic engagement. Putting money behind a post with zero likes is wasted. Wait 2–4 hours after posting, see what's resonating, then boost the winner.

  • Budget: $10–$20/day per boosted post; run each boost for 3–5 days max
  • Targeting: Ages 35–60, interests: freestyle music, Latin R&B, TKA, Judy Torres, Shannon; geo: Wesley Chapel 50-mile radius + NYC metro, NJ, Miami, Orlando, Puerto Rico
  • Format: Boost video posts or Reels over static images — video gets more reach per dollar for this audience
  • Test bilingual copy: Run one English version and one Spanish version; whichever gets better CTR, put more budget there
  • Stop boosting 24 hours before the event — day-of organic content (crowd arriving, venue energy) will outperform any ad
Facebook Retargeting — The Only Cold-to-Warm Bridge

Anyone who visited blackmarker.tv/FE3 and didn't buy is a warm lead. This is the highest-ROI paid spend available.

  • Requires: Meta Pixel installed on blackmarker.tv/FE3 — confirm this is active
  • Audience: Page visitors from the last 14 days who did not reach the confirmation/thank-you page
  • Ad message: Urgency-focused — "Still thinking about it? $10 early bird ends April 20th." → Post April 20: "It's $15 now — last chance before April 25th." → Final days: "Stream goes live tonight — blackmarker.tv/FE3"
  • Budget: $5–$10/day; small pool of warm visitors, doesn't need much spend
  • Format: Single image or short video; keep it simple and direct
⚠ If the Meta Pixel is NOT installed on blackmarker.tv/FE3, skip retargeting and put all budget into boosting organic posts instead.

Goal: As a one-man operation, every minute you spend manually copy-pasting and scheduling is a minute you're not spending on production, relationships, or the next event. This plan automates everything that can be automated — and makes the manual stuff as fast as possible for what can't.
01
Copy-to-Clipboard Buttons
Built directly into this document — zero setup required
Live Now

Every post in the Post Copy tab has a one-click copy button. Click it — the full post text is in your clipboard. Open Facebook, paste, attach your image, post. No scrolling, no selecting text, no typos.

This covers everything that cannot be automated — Facebook group posts, Nextdoor, WhatsApp group chats, YouTube comments. The copy button cuts the friction on all of them.

What will always need a human touch (Facebook ToS + platform rules):

• Facebook Group posts — the API does not allow automated posting to groups. Must be done by a real person.
• Nextdoor posts — no external API; manual only.
• WhatsApp group messages — no broadcast API for group chats; activate through buyers sharing organically.
• YouTube comments — technically possible but flagged as spam; do manually.

For all of these, the copy button is your best tool. One click, paste, done.
02
Scheduling Layer — Set It and Walk Away
Connect your Facebook Page and Instagram once — schedule all posts in one sitting
This Week

Instead of logging in every day to post, you sit down once, schedule everything through April 25th, and let the tool post automatically at the right time. For a one-man band, this is the highest-leverage move you can make right now.

🔄
Buffer — Recommended Starting Point
buffer.com — Free tier supports 3 channels and 10 scheduled posts at a time. Connect your Facebook Page and Instagram account. Copy the post text from this document using the copy buttons, paste into Buffer's composer, set the date and time, and schedule. It posts automatically. The Essentials plan ($6/mo) removes the 10-post queue limit and is worth it for a campaign like this.
⚙
Make (formerly Integromat) — More Power
make.com — Free tier available. More powerful than Buffer — can trigger posts based on conditions, connect multiple services, and handle logic. For example: post goes live on Facebook automatically AND sends you a Slack/text notification AND logs the post in a Google Sheet. Steeper learning curve than Buffer but much more flexible. Recommended for Tier 3 integration.
📅
Meta Business Suite — Free, No Third Party
business.facebook.com — Facebook's own free scheduling tool. Schedule posts directly to your Facebook Page and Instagram from one dashboard. No third-party required. Less polished than Buffer but costs nothing and has no post limits. Good option if budget is zero. Copy from this doc → paste into Meta Business Suite → schedule.
Buffer Setup — Step by Step
  1. Go to buffer.com and create a free account
  2. Click Connect a Channel → select Facebook Page → log in with your Facebook credentials → select the BlackMarker.TV page
  3. Connect Instagram the same way (requires a Business/Creator account linked to the Facebook Page)
  4. Go to the Post Copy tab in this document → click the Copy button on the first post
  5. In Buffer, click Create Post → paste the text → attach your image → set the date and time from the campaign schedule → click Schedule
  6. Repeat for all posts — you can do all of Phase 1–5 in one sitting (takes about 30–45 minutes)
  7. Buffer posts automatically at the scheduled times — you're done until the day-of
03
Direct API Automation — Post From This Document
Schedule and publish to Facebook directly from the Post Copy tab with a date/time picker and one button
Next Phase

The end goal: every post in the Post Copy tab gets a Schedule to Facebook button with a date/time picker. You set the time, click Schedule, and a Make webhook receives the post content and publishes it to your Facebook Page automatically at the right moment — without you touching Facebook at all.

🔑
Meta Graph API — What Makes It Work
Facebook's Graph API allows you to publish posts to a Page programmatically. You need: (1) a Facebook Developer account, (2) an App created in the Meta Developer Console, (3) a long-lived Page Access Token for the BlackMarker.TV page. Once you have the token, posting is a single API call. The token lasts 60 days and can be refreshed.
⚡
Make Webhook — The Bridge
A Make scenario watches a webhook URL. When this document sends post content + a scheduled time to that URL, Make receives it and either posts immediately or schedules it via the Facebook Graph API module. No server required — Make handles everything in the cloud. Free tier supports 1,000 operations/month which is more than enough for an event campaign.
📄
What Gets Built Into This Document
Each post card gets a date/time picker and a "Schedule to Facebook" button. Click the button → the document sends the post text, image note, and scheduled time to the Make webhook → Make posts it to the BlackMarker.TV Facebook Page at exactly that time. You never have to open Facebook to post a main page post again.
Setup Steps for Tier 3
  1. Create a Facebook Developer account at developers.facebook.com
  2. Create a new App → choose Business type → add the Pages API product
  3. Generate a User Access Token with pages_manage_posts and pages_read_engagement permissions
  4. Exchange it for a long-lived Page Access Token (valid 60 days) for the BlackMarker.TV page
  5. Create a Make account at make.com → build a new scenario with a Webhooks → Custom Webhook trigger
  6. Add a Facebook Pages → Create a Page Post module — paste in your Page Access Token and map the incoming post content from the webhook
  7. Add a sleep/delay module if scheduling for a future time, or use Make's built-in scheduling
  8. Copy the webhook URL from Make → we add it to this document → the Schedule buttons go live
⚠ Instagram note: Instagram automation via API requires a Professional/Business account and content approval flow. Images must be hosted at a public URL before posting — they can't be uploaded from a local file. Plan to host event graphics on BlackMarker.tv or an S3 bucket for automated Instagram posts.
