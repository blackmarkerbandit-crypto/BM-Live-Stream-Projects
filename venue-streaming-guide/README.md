A comprehensive knowledge base for **venue owners**, **AV technicians**, and **event planners** looking to add or improve live streaming capabilities at their venues.

## Who This Is For

- **Venue Owners** — Understand what's needed, what it costs, and how to evaluate vendors or build in-house capability.
- **AV Technicians** — Deep-dive into gear selection, signal flow, encoding settings, and troubleshooting.
- **Event Planners** — Plan hybrid (in-person + streamed) events with confidence, including ticketing and legal considerations.

## How to Use This Guide

Each section contains standalone guides you can read in any order. Start with the topic most relevant to your current needs. Cross-references between sections are linked where applicable.
